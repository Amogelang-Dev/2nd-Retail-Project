# 🌶️ Chabis Hotness - Bold Flavors & Fiery Passion

**Chabis Hotness** is a dynamic e-commerce web application built for a South African gourmet chilli sauce brand. The platform provides a seamless end-to-end shopping experience, from browsing unique flavor profiles to secure online payment processing.

---

## 📖 Description
This site serves as the digital storefront for Chabis Hotness, a brand dedicated to stirring up bold flavors in every jar. It features a vibrant, red-themed aesthetic that reflects the heat of the products. Users can explore various flavors like Sweet Chillie and Lemon & Herb, manage a virtual shopping cart, and complete purchases through an integrated payment gateway.

## 🛠️ Technologies Used
* **ASP.NET Core MVC (Razor):** Server-side rendering and site structure using `.cshtml` templates.
* **C#:** Backend logic for cart management and payment verification.
* **JavaScript (Async/Fetch):** Used for real-time "Add to Cart" functionality and handling Paystack API responses.
* **Paystack API:** Integrated for secure, local South African payment processing (ZAR).
* **CSS3:** Internal and external styling featuring responsive layouts and brand-specific color palettes.
* **Font Awesome:** Integrated for social media and UI iconography.

## ✨ Key Features
* **E-commerce Workflow:** Complete shopping journey including product selection, size variations, and a persistent cart.
* **Secure Checkout:** Integrated **Paystack** popup for safe transactions using credit cards or EFT.
* **Flavor Profiles:** Detailed product pages highlighting organic ingredients for each sauce variety.
* **Transaction Status:** Dedicated **Success** and **Failure** pages to provide clear feedback after payment attempts.
* **Brand Storytelling:** An "About Us" section that highlights the founder's journey and company mission.

## ⌨️ Navigation & Shortcuts
* **Header Navigation:** Quick access to **Home**, **Products**, **Shop**, and **Contact**.
* **Cart Management:** Direct links to view the cart and proceed to checkout from any product interaction.
* **External Links:** One-click access to the brand's Facebook, Instagram, and WhatsApp for direct customer support.

## 🏗️ The Build Process
1.  **Architecture:** Utilized the Model-View-Controller (MVC) pattern to separate business logic from the UI.
2.  **Product Integration:** Developed a dynamic product catalog with size-based pricing logic.
3.  **State Management:** Implemented a functional shopping cart using JavaScript and server-side verification.
4.  **Payment Gateway:** Configured Paystack to handle ZAR currency and unique transaction references.
5.  **Feedback Loop:** Created refined UI views for payment confirmation and error handling to ensure a smooth user journey.

## 💡 What I Learned
* **API Integration:** Gained hands-on experience connecting a web app to a third-party payment provider (Paystack).
* **Asynchronous Logic:** Mastered using `fetch` with `RequestVerificationToken` for secure, non-refreshing cart updates.
* **E-commerce Security:** Learned the importance of server-side total recalculation to prevent client-side price manipulation.

## 🚀 Future Improvements
* **User Accounts:** Implement ASP.NET Identity for customer profiles and order history.
* **Database Integration:** Migrate product and order data to a SQL database for better scalability.
* **Admin Dashboard:** Create a secure portal for the owner to manage inventory and view sales analytics.
* **SEO Optimization:** Enhance meta tags and structured data to improve search visibility for South African gourmet foods.
